GetRawTransactionAsync(String, Boolean, String)

Retrieves a transaction given a transaction hash in either simple or verbose form.

Declaration
public Task<TransactionModel> GetRawTransactionAsync(string txid, bool verbose = false, string blockHash = null)
Parameters
Type Name Description
System.String txid

The transaction hash.
System.Boolean verbose

True if verbose model wanted.
System.String blockHash

The hash of the block in which to look for the transaction.
Returns
Type Description
Task<TransactionModel>

A TransactionBriefModel or TransactionVerboseModel as specified by verbose. null if no transaction matching the hash.
Remarks

When called with a blockhash argument, getrawtransaction will return the transaction if the specified block is available and the transaction is found in that block. When called without a blockhash argument, getrawtransaction will return the transaction if it is in the mempool, or if -txindex is enabled and the transaction is in a block in the blockchain.

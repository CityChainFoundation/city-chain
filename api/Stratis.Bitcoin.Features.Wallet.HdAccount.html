Parameters
Type Name Description

The network these addresses will be for.
System.Int32 addressesQuantity

The number of addresses to create.
System.Boolean isChange

Whether the addresses added are change (internal) addresses or receiving (external) addresses.
Returns
Type Description
IEnumerable<HdAddress>

The created addresses.
Remarks

The name given to the account is of the form "account (i)" by default, where (i) is an incremental index starting at 0. According to BIP44, an account at index (i) can only be created when the account at index (i - 1) contains at least one transaction.

Class ScriptToAddressLookup

This class is used to identify a HdAddress given a Script (scriptPubKey).

Inheritance

ScriptToAddressLookup

Namespace: Stratis.Bitcoin.Features.Wallet

Syntax
public class ScriptToAddressLookup : object
Remarks

Being able to map scripts to addresses allows the wallet to identify the corresponding transactions (see Transactions) that pay to those addresses regardless of the type of script being used. The methods are virtual so that the functionality can be overridden as required to support additional script types.

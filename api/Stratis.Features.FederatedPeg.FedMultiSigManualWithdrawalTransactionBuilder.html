BuildTransaction(List<Recipient>, Key[])

Builds a withdrawal transaction from the federation multisig to the given recipients. Determines which coins from the federation wallet to spend for the total amount being sent to recipients. Calculates the fee based on the wallet fee policy. Sets the change address to be the multisig redeem script hash. Signs the transaction using the given private keys.

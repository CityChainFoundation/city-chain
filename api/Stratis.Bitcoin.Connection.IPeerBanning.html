Contract for network peer banning provider.

Namespace: Stratis.Bitcoin.Connection
Assembly: Stratis.Bitcoin.dll
Syntax
public interface IPeerBanning
Remarks

Peers are banned for seconds (default is 24h), this value can change using configuration (-bantime=[seconds]).

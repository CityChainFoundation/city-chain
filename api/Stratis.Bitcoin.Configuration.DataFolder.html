Class DataFolder

Contains path locations to folders and files on disk. Used by various components of the full node.

Inheritance
System.Object
DataFolder
Namespace: Stratis.Bitcoin.Configuration
Assembly: Stratis.Bitcoin.dll
Syntax
public class DataFolder : object
Remarks

Location name should describe if its a file or a folder. File location names end with "File" (i.e AddrMan[File]). Folder location names end with "Path" (i.e CoinView[Path]).

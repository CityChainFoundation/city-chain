BuildGetHeadersPayload()

Builds the GetHeadersPayload.

Declaration
protected override GetHeadersPayload BuildGetHeadersPayload()
Returns
Type Description
GetHeadersPayload

The GetHeadersPayload instance. If the node is a gateway and the peer is not whitelisted, return null (gateways can sync only from whitelisted peers). If the peer can serve PH, GetProvenHeadersPayload is returned, otherwise if it's a legacy peer but it's whitelisted, GetHeadersPayload is returned. If the attached peer is a legacy peer and it's not whitelisted, returns null.

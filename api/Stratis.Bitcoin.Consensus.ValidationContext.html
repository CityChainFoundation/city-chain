Class ValidationContext

A context that is used by consensus that is required for validation, in case validation failed the Error property will be set. It is used when a new block is downloaded or mined.

BanDurationSeconds

If the block validation failed with a ConsensusError that is considered malicious the peer will get banned. The ban, unless specified otherwise, will default to BanTimeSeconds.

Declaration

Property Value
Type Description

Remarks

Setting this value to be BanDurationNoBan will prevent the peer from being banned. Setting this value to be BanDurationDefaultBan will default to BanTimeSeconds.

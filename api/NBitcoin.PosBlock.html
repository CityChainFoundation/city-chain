Gets the block's coinstake transaction or returns the coinbase transaction if there is no coinstake.

Returns
Type Description
Transaction

Coinstake transaction or coinbase transaction.
Remarks

In PoS blocks, coinstake transaction is the second transaction in the block.

In PoW there isn't a coinstake transaction, return coinbase instead to be able to compute stake modifier for the next eventual PoS block.

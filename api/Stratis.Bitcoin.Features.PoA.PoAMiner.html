Class PoAMiner

Mines blocks for PoA network. Mining can happen only in case this node is a federation member.

public class PoAMiner : object, IPoAMiner, IDisposable
Remarks

Blocks can be created only for particular timestamps- once per round. Round length in seconds is equal to amount of fed members multiplied by target spacing. Miner's slot in each round is the same and is determined by the index of current key in GetFederationMembers()

RpcBitcoinImmutableTests

These tests share a test fixture that creates a 101 block chain of transactions to be queried via RPC. This transaction chain should not be modified by the tests as the tests here assume the state of the chain is immutable. Tests that require modifying the transactions should be done in RpcBitcoinMutableTests and set up the chain in each test.
